## Quick Start

### Installation

```bash
npm install justgage
```

### Basic Usage

```javascript
import { JustGage } from 'justgage';

const gauge = new JustGage({
  id: 'my-gauge',
  value: 75,
  min: 0,
  max: 100,
});
```
